# Look Template

The Show Look block authored at Step 5 and locked verbatim. At Step 8 it is pasted unchanged as Bible §1 (see `bible-template.md`) and applied by every prompt. Fill from locked decisions only.

```
### 1. The Look (SHOW LOOK — applied by every prompt)
- Aspect ratio: 16:9 (fixed show-wide — the only supported ratio, applied as an API
  PARAMETER at render time per Bible §3H; not a creative fork. Anamorphic LENS character,
  if wanted, is a Lens/stock entry, not a ratio)
- Lens/stock: [grain, halation, flares, DoF behavior; warm/clinical]
- Color grade (SHOW LOOK default): [name + temp + highlight/shadow bias + saturation;
  name the accent color allowed to pop]
- Optional locked trims: [named scene/sequence Look variants only — e.g. "night-interior
  cooler trim"; per-shot freestyle grading is not a thing]
- Lighting — canonical states ONLY: [STATE: Kelvin, shadow behavior, intended use] × 3–4
- Sound: [one recurring score idea + arrangements; ambient bed; foley; dialogue approach
  + named `@*_vo` handles from `generateVoiceAnchors`]
- Tone north-star: [one sentence]
```

The Look is TEXT: it is pasted into every prompt's global notes, never occupies a reference slot, and never appears in the Bible's §2 material list. A "style_board" entry is a category error.

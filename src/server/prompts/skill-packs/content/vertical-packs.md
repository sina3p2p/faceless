## Vertical focus (shorthand)

Use **`style-modes-01-15.md`** for the full skill map (cinematic, 3D CGI, cartoon, comic, fight, motion-design, e-com, anime, product 360, music, social, brand, fashion, food, real estate). This file is a **short** default bundle:

- **Default** — one beat, clear camera, no competing ideas.
- **Cinematic** — depth, motivated light, **restraint**; move for story, not for flair alone.
- **Social / short-form** — 0–2s read, **one** interrupt or payoff, strong silhouette.
- **Music** — **section** energy; one **hit-friendly** accent; no fake bar numbers.
- **E-com** — product surface + **one** interaction; camera supports the read.
- **Brand** — one **metaphor** move; calm, human-scale; trust over tricks.

For anything else (e.g. **food**, **real estate**, **anime**), lean on the numbered style file, not this list alone.

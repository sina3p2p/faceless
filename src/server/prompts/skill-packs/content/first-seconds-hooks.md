## First 2 seconds

Short-form and feed traffic reward **clarity before polish**. Bias the opening so a viewer can answer “what is this about?” before the first cut.

- **One focal change**: a single clear shift (distance, focus plane, or subject gesture) in the first beat—not three micro-events.
- **Read silhouette**: avoid tangled overlap of limbs and background; favor legible shape.
- **Consequence**: cause → effect in one chain (touch, wind, light catch, weight transfer).
- **End state**: second half of the clip can **settle** or **hand off**; don’t start a second premise at 50% of duration.

### Attention levers (combine at most 2 in one short clip)

- **Pattern interrupt** — something breaks expectation (scale, speed, direction, silence→hit).
- **Curiosity gap** — show **part** of the answer; the motion teases the rest.
- **Contrast** — still→fast, dark→light, wide→tight, quiet→loud (as **motion/light**, not on-screen text).
- **Movement in frame** — peripheral motion in the first frame where policy allows; avoid a static wide with nothing to track.

### Hook technique menu (pick one primary for the open)

| Move | I2V phrasing idea |
|------|-------------------|
| **ECU → wide reveal** | Open on macro detail; one cut or move to **context**; huge scale read. |
| **Black / dark → light burst** | Start underexposed; **one** motivated light or flare enters; settle exposure. |
| **Reverse or float** | Unnatural **slow** roll of an element for 1s, then real physics. |
| **Scale surprise** | Treat a small object as **vast** (macro) or reverse; reveal true scale. |
| **Silence / still → hit** | Hold environment-only or near-still, then **one** motion or light hit. |
| **Color / grade shift** | One **motivated** temperature or saturation swing (practical, flag, time-of-day). |
| **Speed into frame** | Subject or camera **enters** with clear vector; not centered mush. |
| **Rack focus** | **One** plane transfer; background or face locks—no double rack. |
| **Geometry** | Strong **lines** and **asymmetry**; light carves shape; horizon motivated. |
| **Eyes / face lock** | Close, then **one** head/eye move to **direct** attention. |
| **Tilt / roll then level** | Short disorientation, then **stabilize** to readable horizon. |
| **Scale in space** | Figure vs void (tiny in huge or huge in small) with **one** read. |

Avoid on-screen text, watermarks, and UI. Keep language **physical and measurable**—the model already has the image.
